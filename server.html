二、Win10 無法遊玩天堂（作業系統阻擋）

請依以下步驟進行設定：

步驟一：
右鍵點選遊戲啟動檔 → 內容 → 相容性
勾選「以相容模式執行此程式」→ 選擇 Windows 7
勾選「高 DPI 設定替代」→ 套用
建議勾選「以系統管理員身分執行此程式」
※ 若未設定 DPI，畫面可能只顯示左上角
※ 若未選擇相容模式，會出現「無法使用此伺服器」

步驟二：
將螢幕解析度調整為 1280×720 或 1024×768

步驟三：
若仍無法進入遊戲，請對以下檔案按右鍵 → 內容 → 安全性 → 設為「完全控制」
Login.dll、Login.exe、TW13081901.bin、TW13081901.pak

四、無法啟動遊戲（顯示錯誤或閃退）

請確認是否有以下檔案殘留於天堂資料夾，如有請移除：
dgVoodoo、D3D8.dll、D3D9.dll、D3DImm.dll、DDraw.dll
★ 若您系統為 Win7 或顯卡過舊，這些檔案可能導致無法登入遊戲。

補充說明：
dgVoodoo 是一款提升老舊 DirectX 遊戲相容性的補丁工具，可協助遊戲在 Windows 10/11 下正常運行。
若顯卡或晶片過舊不支援，可嘗試：

  • 更新顯卡驅動程式
  • 重新安裝最新版 DirectX
  • 或直接移除 dgVoodoo 檔案恢復原始狀態

五、CreateSurface Failed! 88760234 錯誤閃退解決方法

若開啟遊戲時遇到「createsurface failed! 88760234」導致選完伺服器後閃退：
這是顯示卡未正確強制縮放不支援 DPI 的軟體或遊戲導致。
※ 若未出現此訊息，請勿隨意修改相容性，以免原可遊玩反而出錯。
